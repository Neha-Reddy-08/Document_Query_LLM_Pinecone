# Agriculture QA System

A Streamlit app that allows users to ask questions about agriculture in India. The system uses:

- **LangChain** for retrieval-augmented generation (RAG) chains.
- **OpenAI** embeddings and GPT models for understanding and answering questions.
- **Pinecone** as a vector database to store document embeddings.
- **PDF document processing** for context-aware answers.

## Features

- Upload or store government/agriculture documents in PDF format.
- Ask natural language questions about agriculture in India.
- Retrieve precise answers using semantic similarity and LLM reasoning.
